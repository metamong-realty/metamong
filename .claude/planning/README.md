# Planning 디렉토리

작업별 스펙(spec)과 플랜(plan) 문서를 저장하는 디렉토리입니다.

## 구조

```
planning/
├── {브랜치명}-spec.md    # 요구사항 명세
├── {브랜치명}-plan.md    # 구현 계획
└── README.md            # 이 파일
```

## 사용법

1. **새 기능 개발 시**
   - "스펙 작성해줘" → `{브랜치명}-spec.md` 생성
   - "플랜 작성해줘" → `{브랜치명}-plan.md` 생성

2. **파일명 규칙**
   - 현재 브랜치명을 기준으로 자동 생성
   - 예: `feature/user-auth-spec.md`, `feature/user-auth-plan.md`

3. **팀 협업**
   - 작업 히스토리 추적 가능
   - 의사결정 과정 문서화
   - 나중에 참고할 수 있는 설계 문서
